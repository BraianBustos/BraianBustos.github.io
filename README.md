# BraianBustos.github.io
Proyecto-Resto-Bar
El programa es un menú enfocado en un negocio de comidas rápidas, en el que el usuario selecciona dentro de una lista de opciones, qué le 
gustaría comer y tomar, con un sólo click sobre la opción deseada. Esta acción realiza los cálculos necesarios para que se haga de forma automática
la suma de los gatos que representan las opciones que eligió. Con el botón que se encuentra en la hoja de gastos en el margen inferior derecho
se puede ver ese valor; y confirmar el pedido con el botón que se encuentra abajo de esta hoja. Al confirmarlo, se envía un formulario con
el mismo a la contraparte que lo recibe en la administración del restobar. 

La próxima versión del programa va a incluir una opción para corregir un pedido erróneo, otra para dividir el total de gastos entre la cantidad 
de comensales y una mayor optimización y estructuración del código.
